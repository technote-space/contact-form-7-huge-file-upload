# Contact Form 7 huge file upload
Contact Form 7 の機能を拡張するプラグインです。
通常アップロードできないような大容量のファイルをアップロードできるようになります。

# 要件
- PHP 5.6 以上

# 導入手順
1. ZIPダウンロード  
2. wp-content/plugins に展開  
3. 管理画面から有効化  
4. Contact Form 7 のフォーム編集画面で対象のファイルに「fileupload」クラスを追加
